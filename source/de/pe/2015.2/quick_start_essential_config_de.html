Im folgenden Abschnitt werden allgemeine Konfigurationsaufgaben beschrieben, die mit Puppet Enterprise verwaltet werden können. Die folgenden Einzelschritte sind eine sehr gute Einführung in die Eigenschaften von PE.

NTP Schnellstart Anleitung

NTP ist einer der kritischsten und zugleich mittels PE einfach zu konfigurierenden Services. Folgen Sie dieser Anleitung um auf allen von PE kontrollierten Systemen eine einheitliche Zeitkonfiguration zu haben.

DNS Schnellstart Anleitung

Diese Anleitung erklärt alle notwendigen Schritte um einen DNS Nameserver mittels PE zu verwalten. Ein Nameserver ermöglicht die einfache Übersetzung von durch Menschen lesbare Internet Adressen innerhalb des Browsers (als Beispiel, google.de) in eine IP Adresse zu übersetzen, die durch einen Computer gelesen werden kann.

SSH Schnellstart Anleitung

Diese Anleitung beinhaltet alle notwendigen Schritte um die Konfiguration von SSH innerhalb der PE Umgebung mittels eines existierenden Puppet Forge Moduls zu automatisieren.

Sudo Schnellstart Anleitung

Das Management von Sudo auf ihren Agenten ermöglicht es die Berechtigungen von Benutzern auf ihrem System zu kontrollieren. Diese Anleitung beinhaltet alle notwendigen Schritte um die Konfiguration von Sudo mittels eines Puppet Forge Moduls in Verbindung mit einem einfachen Moduls, welches Sie selber ertstellen werden, zu ermöglichen.

Firewall Schnellstart Anleitung

Befolgen Sie die Einzelschritte in dieser Anleitung um die Konfiguration von Firewall Regeln mittels eines einfachen Moduls zu ermöglichen, welches diese Regeln bestimmt.
