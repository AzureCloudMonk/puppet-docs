¡Bienvenido a la guía de inicio rápido de Puppet Enterprise!

Ya sea que estes configurando PE para un despliegue en producción o que desees aprended algunos conceptos fundamentales sobre manejo de confifuración con Puppet Enterprise, esta serie de guías te proveeran la información necesaria para ponerte en marcha. Te guiaremos a través de la configuración de un despliegue monolítico (todo en un nodo) y te demostraremos como automatizar algunas de las tareas básicas que los administradores de sistemas performan regularmente.

Presentación de las guías de inicio rápido * nix

Las siguientes guías presentan las tareas en el orden el cual probablemente las realizarías. Por favor, revisa la seccion de prerequisitos de cada guía para asegurarte que tu entorno esta configurado como se espera para realizar los pasos como estan definidos.

1. Instalar un despliegue Monolítico de Puppet Enterprise

La configuración básica de PE instala Puppet master, la consola PE y PuppetDB en un único nodo, una maquina *nix. Esta guía también revisa los requisitos previos necesarios para una instalación PE.

2. Instalar el Agente de Puppet

Esta gúia instala un Puppet agent que regularmente que regularmente tira de los catálogos de configuración de un Puppet master y los aplica localmente. Estas instrucciones incluyen los pasos para firmar el certificado de cada agente en la consola.

3. Instalar un Módulo

Esta guía instala un module de Puppet, El medio principal mediante PE configura y gestiona nodos. Miles de modules están disponibles para su uso en Puppet Forge.

4. Agregar classes

Esta guía crea un nuevo grupo de nodos, aplicamo un fragmento llamado de código de Puppet - mejor conocido como class — al grupo de nodos.

5. Dar acceso a los usuarios

Esta sección le presenta el control de acceso basado en roles. Se creará una nueva role para el usuario y se concederá el permisos para trabajar con el grupo de nodos que creó anteriormente.

6. Escribir un module

Por último, escribirá un módulo para ayudarle a familiarizarse con los módulos Puppet y el desarrollo de módulos.

Inténtalo tú mismo

  1. Instalar PE
  2. Instalar Puppet Agent
  3. Instalar un module
  4. Agregar classes
  5. Dar acceso a los usuarios
  6. Escribe tu propio módulo

Próximos pasos

Una vez que haya completado la guía de inicio rápido, recomendamos los siguientes pasos:

  • Aprenda más sobre cómo trabajar con Puppet Forge para encontrar los modules necesarios
  • Utilice el método de roles y perfiles para crear una configuración completa del sistema y almacenarla en un repositorio de control
  • Gestionar código Puppet utilizando las herramientas incluidas en PE y el repositorio de control
  • Clasificar nodos utilizando grupos y reglas
