Cómo funciona la instalación en modo texto

Cuando ejecuta el instalador de Puppet Enterprise en modo de texto, pasa un archivo de configuración (pe.conf), que contiene los valores necesarios para instalar PE, al instalador.

La instalación en modo texto requiere que ejecute el instalador con el fichero pe.conf en cada nodo en el cual está instalando un componente PE. En otras palabras, para una instalación monolítica, usted ejecuta un instalador con pe.confen un nodo, para una instalación fragmentada, usted ejecuta el instalador con pe.conf en tres nodos.

  • Instrucciones de instalación monolíticas en modo texto
  • Instrucciones de instalación en modo texto dividido

Antes de comenzar una instalación, revise, seleccione una arquitectura PE.

El fichero pe.conf

Un fichero pe.conf es un formato de fichero HOCON que declara parámetros y valores necesarios para instalar y configurar PE. Por ejemplo, para una instalación monolítica, sólo necesita dos pares clave / valor: 

"console_admin_password": "puppet"
"puppet_enterprise::puppet_master_host": "%{::trusted.certname}"

Consulte pe.conf referencia de parámetrospara más detalles para configuración de parámetros con pe.conf y para una lista completa de parámetros obligatorios y opcionales.

Obteniendo un fichero pe.conf

Tiene algunas opciones para crear u obtener un fichero pe.confpara uso durante las instalaciones.

  • Utilice el fichero pe.conf proporcionado en el directorio conf.den el tarball instalador de PE.

    Truco:En la mayoría de los casos, para una instalación monolítica, usted puede utilizar el fichero de ejemplo pe.conf sin realizar ningún cambio.

  • Cuando el instalador lo solicite, seleccione la opción de instalación en modo texto. Esto abre el editor predeterminado con el fichero de ejemplo pe.conf, el cual usted puede editar. La instalación continuará utilizando pe.conf después de salir del editor.
  • Utilice el instalador basado en web para crear un fichero pe.conf. Después de ejecutar el instalador basado en web, puede encontrar el archivo en /etc/puppetlabs/enterprise/conf.d. También puede descargar el archivo siguiendo el enlace proporcionado en la página de confirmación del instalador basado en web.

Para obtener una lista completa de parámetros obligatorios y opcionales, pe.conf referencia de parámetros.
