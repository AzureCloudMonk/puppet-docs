--- title: "Hiera: Actualiza las llamadas clásicas de la función Hiera" ---

Las funciones hiera, hiera_array, hiera_hash y hiera_include están obsoletas y serán eliminadas en Puppet 6.

La función lookup es un reemplazo completo para todos estos:

Función Hiera Equivalente llamada lookup
hiera('secure_server') lookup('secure_server')
hiera_array('ntp::servers') lookup('ntp::servers', {merge => unique})
hiera_hash('users') lookup('users', {merge => hash}) or lookup('users', {merge => deep})
hiera_include('classes') lookup('classes', {merge => unique}).include

Para asegurarse de que está listo para Puppet 6 y más allá, revise sus módulos de títeres para reemplazar las funciones hiera_* con lookup. Esto es puramente para prepararse para el futuro: puede adoptar todas las nuevas funciones de Hiera 5 sin actualizar estas llamadas de función.

Mientras está revisando, considere refactorizar algún código para usarlo Búsqueda automática de parámetros de clase en vez de funciones — desde lookups automáticos Ahora puede hacer fusiones únicas y hash, el hiera_array y las funciones hiera_hash no son tan importantes como solían ser.
